## Quick Start :: LaTeX users
1. Go to the projects dir
2. Create a tex file
3. Declare the preamble and template `\usepackage{preamble}`, `\usepackage{adkore}`
4. Fill in fields given by template partial, see `sample.tex` in `...\projects\`
